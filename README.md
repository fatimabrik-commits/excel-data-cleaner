# 📊 Excel Data Cleaner

A simple Python tool that cleans Excel files automatically.

## ✨ Features

- Removes duplicate rows
- Removes empty rows
- Removes extra spaces
- Saves the cleaned file automatically
- Creates an output folder

## 🛠 Requirements

- Python 3
- pandas
- openpyxl

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run:

```bash
python cleaner.py
```

Enter your Excel file name, for example:

```
data.xlsx
```

## 📁 Output

The cleaned file will be saved in:

```
output/cleaned_data.xlsx
```
